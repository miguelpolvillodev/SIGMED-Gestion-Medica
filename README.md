# SIGMED - Sistema Integral de Gestión Médica

Sistema de gestión corporativo para clínicas y centros de salud, diseñado para administrar el flujo de pacientes, facultativos y agenda médica de forma eficiente y segura.

## Características Principales

### Backend (Java & Oracle)
* **Arquitectura MVC y DAO:** Código modular y desacoplado siguiendo estándares de la industria.
* **Gestión de Datos:** Persistencia robusta con **Oracle Database** y transacciones JDBC.
* **Lógica de Negocio Avanzada:** Algoritmos de validación para evitar conflictos de agenda (Time Slot Management) y duplicidad de citas.
* **Seguridad:** Control de acceso basado en roles (Administrativo / Médico).

### Frontend (Prototipo UI)
* Diseño de interfaz moderno y responsivo utilizando **HTML5** y **CSS3**.
* Estética corporativa (Azul Médico) enfocada en la usabilidad y claridad visual.

## Tecnologías Utilizadas
* **Lenguaje:** Java SE (JDK 21)
* **Base de Datos:** Oracle Database Express Edition (XE) / SQL
* **Interfaz:** HTML5, CSS3
* **Herramientas:** VS Code, Eclipse, Git
